# System Architecture

The MediaDotGames platform transforms raw news data into structured analytical signals.

Major components:

1. News ingestion
2. Story storage
3. Topic clustering
4. Public interest evaluation
5. Coverage aggregation
6. Visualization

Data Flow:

News Sources
↓
Ingestion Pipeline
↓
Story Database
↓
Clustering Pipeline
↓
Topic Database
↓
Public Interest Evaluation
↓
Coverage Analytics
↓
Visualization
